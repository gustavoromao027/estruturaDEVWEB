Trabalho: Elaboração de estrutura de navegação e template html.


Este trabalho representa a síntese do conhecimento adquirido em sala de aula, onde os conceitos fundamentais de HTML, CSS e Python foram aplicados de maneira integrada. Nosso objetivo foi criar um site dinâmico e funcional composto por cinco páginas, explorando habilidades avançadas de design e interatividade. Além disso, utilizamos um arquivo Excel como banco de dados, demonstrando nossa capacidade de integrar diferentes tecnologias para alcançar um resultado final coeso e inovador. Este projeto não apenas representa nosso comprometimento com a excelência acadêmica, mas também nossa habilidade em aplicar os conhecimentos teóricos em um contexto prático e desafiador..
